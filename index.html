ACTIVIDAD 1 CSS (Valor 10 puntos).

  1. ¿Qué es CSS?

    2. Lenguaje de estilos utilizado para describir la presentación de un documento HTML.

  2. ¿Cuál es la función principal de CSS?

    3. Función principal de CSS. Dar estilo y formato a los elementos HTML, como colores, fuentes, tamaños, márgenes, entre otros, separando el contenido del diseño visual.

  3. ¿Cómo se puede incluir CSS en un documento HTML?

    4. En un documento HTML, CSS se puede incluir mediante una hoja de estilo externa enlazada con link, estilos internos en la etiqueta style, o aplicando estilos en línea con el atributo style dentro de los elementos HTML.

  4. ¿Qué son los selectores en CSS?

    5. Los selectores en CSS son mecanismos que permiten seleccionar elementos específicos del documento para aplicarles estilos, ya sea por su etiqueta, clase, ID o combinaciones más complejas.

  5. ¿Qué son las propiedades y valores en CSS?

    6. Las propiedades en CSS son las características que definen el estilo de un elemento (como color o font-size), mientras que los valores son las definiciones que asignas a esas propiedades (por ejemplo, color: red).

  6. ¿Cómo se aplica el modelo de caja en CSS?

    7. El modelo de caja en CSS se refiere a la forma en que los elementos HTML son representados en el diseño, con cuatro áreas: contenido, relleno (padding), borde, y márgenes. Esto define cómo se calculan los tamaños y posiciones.

  7. ¿Qué es el "cascading" en CSS?

    8. El "cascading" en CSS se refiere al sistema de priorización que determina qué reglas de estilo se aplican cuando varias coinciden con un mismo elemento. Las reglas más específicas o declaradas más abajo en el código tienen prioridad.

  8. ¿Qué son las media queries?

    9. Las media queries son condiciones que permiten aplicar diferentes estilos CSS dependiendo del tamaño de la pantalla o el dispositivo, logrando adaptaciones para diferentes resoluciones.

  9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

    10. Entre los estilos comunes que se pueden aplicar en CSS están el color de texto, tamaño de fuente, márgenes, relleno, bordes, alineación de elementos y fondos.

  10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

    11. Para crear un diseño responsivo con CSS, se usan técnicas como el uso de flexbox, grid, y media queries, que permiten ajustar los estilos y el diseño de los elementos según el tamaño de la pantalla o dispositivo.
